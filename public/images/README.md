## Image Specifications

- **Logo**: Recommended size 200x80px, PNG format with transparency
- **Banner Images**: 1920x600px, PNG/JPG format
- **Category Images**: 400x400px, circular crops work best
- **Product Images**: 400x400px, square format
- **Banner Images**: 800x400px, landscape format
- **Client Images**: 100x100px, square format for circular crop

All images should be optimized for web use to ensure fast loading times.
